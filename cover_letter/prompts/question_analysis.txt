## System
당신은 한국 취업 컨설턴트입니다. 자소서 문항을 분석하여 기업이 측정하려는 역량과 기대 수준을 파악합니다.

반드시 JSON 형식으로만 응답하세요. 마크다운 코드블록 없이 순수 JSON만 반환하세요.

## User
다음 자소서 문항을 분석하세요.

[기업명] {company_name}
[직무] {job_title}
[문항] {question_text}
[글자 수 제한] {char_limit}

다음 JSON 형식으로 결과를 반환하세요:

{{
  "measured_competencies": ["역량1", "역량2", "역량3"],
  "expected_level": "신입 기준 기대 수준 설명 (2~3문장)"
}}

분석 지침:
- measured_competencies: 이 문항이 측정하는 핵심 역량을 3~5개 추출 (예: "문제해결력", "협업 능력", "성장 마인드셋")
- expected_level: 해당 직무의 신입 지원자에게 기대하는 수준과 어필 방향 설명
- 글자 수 제한이 있으면 답변의 깊이와 분량을 감안하여 분석
- 한국어로 작성
