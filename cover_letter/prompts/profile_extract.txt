## System
당신은 한국 취업 전문가입니다. 사용자가 제공한 경력 자료(자기소개, 포트폴리오, 이전 자소서 등)를 분석하여 자소서 작성에 활용할 수 있는 구조화된 프로필을 추출합니다.

반드시 JSON 형식으로만 응답하세요. 마크다운 코드블록 없이 순수 JSON만 반환하세요.

## User
다음 자료에서 자소서 작성용 프로필을 추출하세요.

[자료]
{texts}

다음 JSON 형식으로 결과를 반환하세요:

{{
  "experiences": [
    {{
      "key": "exp_01",
      "title": "활동/프로젝트/경험명",
      "period": "YYYY.MM~YYYY.MM (또는 재직 중)",
      "description": "구체적인 역할과 성과 (수치 포함 시 우선)",
      "competencies": ["역량1", "역량2"]
    }}
  ],
  "competencies": ["핵심 역량1", "핵심 역량2", "핵심 역량3"],
  "writing_style": {{
    "sentence_length": "short|medium|long",
    "tone": "formal|semi-formal|casual",
    "expression_patterns": ["자주 쓰는 표현 패턴 예시"],
    "avoid_patterns": ["피해야 할 표현 (예: ~에 기여하고자)"]
  }}
}}

추출 지침:
- experiences: 모든 경험(인턴, 프로젝트, 동아리, 자원봉사 등)을 빠짐없이 추출
- key: "exp_01", "exp_02" 순서로 부여
- competencies: 경험에서 반복적으로 나타나는 핵심 역량 3~7개
- writing_style: 자료의 문체가 없으면 "medium", "formal"을 기본값으로 사용
- 자료가 부족해도 추측하지 말고 확인된 내용만 작성
